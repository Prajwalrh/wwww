DeepFake Detection using ResNeXt and LSTM

Team and Major project || Django || Deep learning || Google Collab || Python || Pytorch

I worked on this major project as a team member for nearly three to four months, starting in January 2024. The project began with the crucial task of collecting a comprehensive dataset from various platforms, followed by meticulous preprocessing to prepare the data for training our models. We utilized ResNeXt, a powerful neural network architecture, to extract features from video frames, which was instrumental in identifying subtle visual cues. To capture and analyze sequential dependencies across frames, we implemented an LSTM (Long Short-Term Memory) network, enhancing the accuracy of deepfake detection. The culmination of our efforts was the development of a Django web application, providing users with an intuitive interface to upload videos and verify their authenticity. This application enables users to easily check whether a video is real or fake, leveraging the power of our trained models. A Chatbot Application using Android Studio

Team project || Java || Android Studio || Palm2 API

As part of a collaborative team, I played a pivotal role in developing an advanced Android application that generates intelligent, context-aware responses to user queries using the Google PaLM2 API. Built in Android Studio with Firebase for real-time data management and user authentication, the app offers a secure and seamless user experience. Our project aimed to replicate much of the functionality of conversational AI models like ChatGPT, achieving over 70% of its capabilities. This effort involved rigorous testing and iterative improvements, resulting in a highly functional and reliable application. The project not only enhanced our technical skills but also provided valuable experience in teamwork and problem-solving, showcasing our ability to innovate in AI and mobile development.

Online Banking System

Team project|| HTML || CSS || JavaScript || PHP || SQL

The Online Banking System was a team project where we integrated the frontend, backend, and database to create a comprehensive application. The system featured login and sign-up functionality for both admin and user accounts. We used PHP for the backend and SQL for database interactions. Users could register for online banking, manage their accounts, and perform transactions, all of which were visible to the admin for monitoring and management. This project provided valuable experience in full-stack development, emphasizing the integration of different technologies to deliver a functional and secure banking solution.
